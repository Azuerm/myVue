mapActions辅助函数 mapActions是把actions中的方法提取出来，映射到组件methods中
actions: { changeCountAction (context, num) { setTimeout(() => { context.commit('changeCount', num) }, 1000) } }
methods: { handleChange (n) { this.$store.dispatch('changeCountAction', n) } } 简化： import { mapActions } from 'vuex' methods: { ...mapActions(['changeCountAction']) } 调用：this.changeCountAction(n)
