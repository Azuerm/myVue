辅助函数-mapMutations mapState映射状态；mapMutations映射方法
mapMutations把位于mutations中的方法提取出来，映射到组件methods中
mutations: { subCount (state, n) { state.count -= n } }
methods: { subCount (n) { this.$store.commit('subCount', n) } }

简写为：

import { mapMutations } from 'vuex' methods: { ...mapMutations(['subCount']) } 调用： this.subCount(10)
methods: { ...mapMutations(['subCount']), handelSub (n) { this.subCount(n) } } 或者 页面直接用
