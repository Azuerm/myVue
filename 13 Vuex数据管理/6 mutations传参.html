mutations必须是同步的(便于监测数据变化，记录调试) mutations传参语法： 提交mutation可以传递参数的 => this.$store.commit('xxx方法名', 参数) 注意：mutation参数有且只能有一个，如果需要多个参数，包装成一个对象或者数组
1.提供mutation函数(带参数-提交载荷) 2.页面中提交调用mutation

1.提供mutation函数

mutations: { addCount (state, n) { state.count += n } }

2.页面中提交调用mutation

methods: { handleAdd (n) { this.$store.commit('addCount', n) } }

mutation参数有且只能有一个，如果需要多个参数，包装成一个对象或者数组

methods: { handleAdd (n) { this.$store.commit('addCount', { count: n, msg: '哈哈' }) } } mutations: { addCount (state, obj) { console.log(obj) state.count += obj.count } }
