# Vue组件通信练习题

## 通信
1. 父组件向 -> 子组件传值
    * 方式1：自定义属性 + props

2. 子组件向 ->  父组件传值
    * 方式1：$emit + 自定义事件

3. 兄弟组件 <-> 之间传值
    * 方式1：创建一个utils/ EventBus.vue, 在接收方监听$on,在发送方触发$emit

4. 非父子组 <-> 件之间传值
    * 方式1：provide 提供数据，inject 接受
